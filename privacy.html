Privacy Policy

Last updated: May 21, 2026

1. Overview

This privacy policy describes how the.lost.broadcast handles information in connection with its use of the TikTok API.

2. Data We Collect

This application does not collect, store, or share personal data from any third parties. The only data processed is content created by the application owner for the purpose of publishing to their own TikTok accounts.

3. TikTok API Data

This application uses TikTok's Content Posting API to publish photos and videos to accounts owned by the application owner. Access tokens are stored locally on the owner's private machine and are never shared or transmitted to any third party.

4. Cookies

This application does not use cookies.

5. Third Parties

This application does not share any data with third parties.
